  HOW TEMPERATURE WORKS:
----------------------------------------------------------------------
  Temperature controls how the model picks the next token (word).

  At each step, the model calculates probabilities for possible
  next tokens. Temperature adjusts those probabilities:

    temp=0.0 → Always picks the highest-probability token (deterministic)
    temp=0.5 → Some variation, mostly follows top probabilities
    temp=1.0 → Spreads probability evenly, more random/creative picks

  You'll see below: at temp=0.0 all responses are similar.
  At temp=1.0 responses are wildly different from each other.
