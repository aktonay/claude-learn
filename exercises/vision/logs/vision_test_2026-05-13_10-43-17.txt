======================================================================
  KEY TAKEAWAYS
======================================================================

  1. IMAGE FORMAT: Images are sent as base64-encoded data in
     the message content array alongside text blocks.

  2. TOKEN COST: Each image costs tokens = (width x height) / 750
     A 1000x1000 image costs ~1333 tokens before the model even reads it.

  3. PROMPT ENGINEERING APPLIES TO IMAGES TOO:
     - Naive: 'Describe this image' -> vague, incomplete
     - Structured: step-by-step methodology -> detailed, accurate
     - Same techniques that work for text work for images

  4. VISION MODELS ON ZAI:
     - glm-5, glm-5-turbo: support images
     - glm-4.5, glm-4.5-air: do NOT support images

  5. LIMITATIONS:
     - Max 100 images per request
     - Max 5MB per image
     - Single image: max 8000px height/width
     - Multiple images: max 2000px each
