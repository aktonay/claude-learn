======================================================================
  KEY TAKEAWAYS
======================================================================

  1. CITATIONS BUILD TRUST:
     Without citations, users can't verify where info comes from.
     With citations, every claim traces back to source material.

  2. TWO WAYS TO GET CITATIONS:
     a) Prompt-based: ask model to cite inline [source: "..."]
        Works with ANY model, ANY API. Parse citations with regex.
     b) Native API (Anthropic only): structured citation objects
        Returns page numbers, document index, exact spans.

  3. CITATION VALIDATION:
     Always verify cited text actually exists in the source.
     Models sometimes hallucinate or paraphrase citations.

  4. DOCUMENT STRUCTURE:
     Wrap documents in XML tags: <document title="...">...</document>
     This helps the model distinguish source from instructions.

  5. PDF PROCESSING:
     Anthropic API: base64 encode PDF, use type="document"
     ZAI API: extract text from PDF first, send as text content
     Both benefit from the same citation prompting techniques.

  6. REAL-WORLD USE CASES:
     - Insurance: satellite imagery fire risk with source tracking
     - Legal: contract analysis with clause citations
     - Medical: literature review with paper citations
     - Education: reading comprehension with text evidence
